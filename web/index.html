Virtual Cavity RF Simulator - Help
How to Use:
  1. Start Simulation: Click "Start Simulation" to begin real-time cavity response calculation
  2. Adjust Parameters: Use the sliders to change RF amplitude, phase, frequency offset, and beam current
  3. Monitor Response: Watch the real-time plots showing cavity voltage, reflected power, and detuning
  4. Record Data: Click "Start Recording" to save simulation data for later analysis
  5. Export Data: Save recorded data as CSV or JSON format
